第8講 / 化学変化と質量

質量保存の法則 ★★☆

化学変化が起こっても、反応の前と後で全体の質量は変わりません。これを「質量保存の法則」といいます。3つの実験を通して、この大事な法則を学んでいきましょう。

1質量保存の法則とは
📖 説明
0:00.0

第7講で「化学反応式は左辺と右辺で原子の数が等しくなる」と学びました。これは別の言い方をすると、反応の前と後で、物質をつくっている原子の数が変わらないということです。

原子の数が変わらないということは、反応の前と後で全体の質量も変わらないということになります。これを「質量保存の法則」といいます。

⚗️ 質量保存の法則

化学変化の前と後で、反応に関係する物質全体の質量は変わらない。

▶ なぜ質量は変わらないの?

化学変化が起こると、原子の組み合わせは変わります。しかし、原子そのものが新しく生まれたり消えたりすることはありません。

たとえば、銅 + 酸素 → 酸化銅 という反応では、Cu と O の組み合わせが変わるだけで、Cu原子の数も O原子の数も反応前と後で同じです。原子1個1個には決まった重さがあるので、原子の数が変わらない以上、全体の質量も変わらないのです。

▶ 注意!質量が変わらないのは「全部を測れば」

ここで大事な注意点があります。質量保存の法則は、反応に関係するすべての物質を測ったときに成り立ちます。たとえば気体が発生する反応で、発生した気体が空気中に逃げてしまうと、てんびんの数値だけ見ると軽くなったように見えます。

でもこれは原子が消えたわけではなく、気体になった原子が容器の外に出ていっただけです。次の単元から、3つの実験を通してこの法則を確かめていきましょう。

★ 覚えよう!
  • 質量保存の法則 …… 化学変化の前後で全体の質量は変わらない
  • 理由:原子の組み合わせが変わるだけで、原子の数は変わらないから
  • 注意:気体が逃げると、てんびんの数値は軽くなる(法則の例外ではない)
  • 密閉容器なら、気体も含めて質量は変わらない

    2沈殿ができる反応 ― 硫酸+水酸化バリウム
    📖 説明
    0:00.0

    1つ目の実験です。うすい硫酸と水酸化バリウム水溶液を混ぜます。両方とも無色透明の液体です。

    2つの液体を混ぜると、白色の硫酸バリウム(BaSO₄)という物質が沈殿します。沈殿とは、水にとけずに底に沈む固体のことです。

    ▶ 反応の式

    うすい硫酸 + 水酸化バリウム → 硫酸バリウム + 水

    (白色の沈殿)

    ▶ 質量はどうなる?

    反応の前と後で、電子てんびんで全体の質量を測ってみると、まったく変わらないことがわかります。これが質量保存の法則です。

    この実験では気体が発生せず、反応してできたものはすべて容器の中にあります。だから、前後の質量がぴったり同じになるのが、目で見て確認しやすいのです。

    💡 豆知識:硫酸バリウム(BaSO₄)はX線を通さないので、レントゲンで胃の形を見るときに「バリウムを飲む」検査で使われています。

    ★ 覚えよう!
    • うすい硫酸 + 水酸化バリウム → 硫酸バリウム(白色の沈殿) + 水
    • 反応の前と後で、全体の質量は変わらない
    • 沈殿…水にとけずに底に沈む固体のこと
    📚 語群 ― タップして選び、説明文の空欄をタップ!
    ✏️ ミニテスト(全3問)
    問1うすい硫酸と水酸化バリウムを混ぜたときにできる白色の沈殿は何か。
    正解はイ:硫酸バリウム。化学式は BaSO₄ です。
    問2うすい硫酸と水酸化バリウムを混ぜると、反応の前と後で全体の質量はどうなるか。
    正解はウ:変わらない。気体が発生せず、すべての物質が容器の中に残るので、質量保存の法則がそのまま成り立ちます。
    問3反応の前後で質量が変わらないのは、原子の組み合わせが変わるだけで原子の数は変わらないからである。
    ○ 正解です。質量保存の法則の根拠です。原子は化学変化で消えたり生まれたりしません。

      3気体が発生する反応 ― 炭酸水素ナトリウム+塩酸
      📖 説明
      0:00.0

      2つ目の実験です。今度は気体が発生する反応です。炭酸水素ナトリウム(NaHCO₃)にうすい塩酸(HCℓ)を加えると、気体の二酸化炭素(CO₂)が発生します。

      密閉容器の中で反応させて、ふたを閉めたままの場合と、ふたを開けた場合で、てんびんの数値はどう変わるでしょうか?

      ▶ 反応の式

      炭酸水素ナトリウム + うすい塩酸 → 二酸化炭素 + 塩化ナトリウム + 水

      ▶ ふたを閉めたまま

      密閉容器でふたを閉めたまま反応させると、発生した二酸化炭素は容器の中に閉じ込められます。だから、反応の前後で全体の質量は変わらない(100.0g → 100.0g)。質量保存の法則がきれいに成り立ちます。

      ▶ ふたを開けると?

      反応が終わってからふたを開けると、容器の中にたまっていた二酸化炭素が空気中に逃げるので、てんびんの数値は気体の分だけ軽くなります(100.0g → 98.5g)。

      でもこれは「質量保存の法則がやぶられた」のではありません。気体になった原子も含めてすべて測れば、ちゃんと質量は変わっていません。「逃げた気体の分だけ、てんびんの数値が減って見えるだけ」なのです。

      ★ 覚えよう!
      • 炭酸水素ナトリウム + 塩酸 → 二酸化炭素 + 塩化ナトリウム + 水
      • 密閉(ふた閉)…気体も容器内 → 質量変わらない
      • 開放(ふた開)…気体が逃げる → 軽くなる(数値上)
      • 気体が逃げただけで、原子は消えていない
      📚 語群 ― タップして選び、説明文の空欄をタップ!
      ✏️ ミニテスト(全4問)
      問1炭酸水素ナトリウムに塩酸を加えたときに発生する気体は何か。
      正解はイ:二酸化炭素。石灰水を白くにごらせるのが特徴です。
      問2密閉容器内で炭酸水素ナトリウムと塩酸を反応させ、ふたを閉めたまま全体の質量を測ると?
      正解はウ:変わらない。発生した二酸化炭素も容器内に残るので、質量保存の法則が成り立ちます。
      問3反応後にふたを開けて全体の質量を測ると、ふたを開ける前と比べてどうなるか。
      正解はイ:軽くなる。発生した二酸化炭素が空気中に逃げるので、その分だけ軽くなります。
      問4ふたを開けて軽くなったということは、原子が消えたということである。
      × 原子は消えていません。気体になって容器の外に出ただけです。気体も含めて全部測れば質量は変わりません。

        4気体が発生する反応 ― 塩化アンモニウム+水酸化ナトリウム
        📖 説明
        0:00.0

        3つ目の実験です。下の図のような特殊な試験管(①)を使います。①は二また試験管といい、容器の中で2つの物質を別々にしておき、傾けて混ぜることができます。

        片側に塩化アンモニウムと水酸化ナトリウムの粉末、もう片側に水を入れ、試験管の口にはゴム風船をつけます。

        試験管を傾けて反応させると、気体のアンモニアが発生し、いったん風船がふくらみます(下図 A)。しかしアンモニアは水にとけやすいので、風船はすぐにしぼみます(下図 B)。

        ▲ ①は二また試験管。塩化アンモニウムと水酸化ナトリウムの粉末を片側、水をもう片側に入れて傾けて混ぜる。A:アンモニアが発生して風船がふくらむ。B:アンモニアが水にとけて風船がしぼむ。

        ▶ 反応の式

        ⚗️ 【重要】反応の式

        塩化アンモニウム + 水酸化ナトリウム → アンモニア + 塩化ナトリウム + 水

        ▶ A・Bの状態をくらべよう

        上図 A の場合 気体のアンモニアが発生し、風船がふくらむ。
        上図 B の場合 しばらくすると発生したアンモニアが水にとけるので、風船がしぼむ。

        しかし、反応前後の質量は変化しない。

        ▶ なぜ風船はしぼむの?なぜ質量は変わらないの?

        風船は一度ふくらんでから、しばらくするとしぼんでいきます。これは、アンモニアが水によくとける性質を持っているため、発生したアンモニアが容器の中の水に少しずつとけて気体の量が減るからです。

        でも、てんびんの数値はAでもBでもどちらも同じ 102.3g のままです。アンモニアが「気体」から「水にとけた状態」に変わっただけで、容器の外には出ていないからです。風船の見た目は大きく変わるのに質量は変わらない ― 原子の数で考えれば当然のことなのです。

        ★ 覚えよう!
        • 塩化アンモニウム + 水酸化ナトリウム → アンモニア + 塩化ナトリウム + 水
        • 反応直後:アンモニアが発生 → 風船がふくらむ
        • しばらく後:アンモニアが水にとける → 風船がしぼむ
        • A・B いずれの時点でも、容器全体の質量は変わらない
        📚 語群 ― タップして選び、説明文の空欄をタップ!
        ✏️ ミニテスト(全4問)
        問1塩化アンモニウムと水酸化ナトリウムを反応させると、何という気体が発生するか。
        正解はエ:アンモニア。化学式はNH₃。鼻にツンとくる刺激臭のある気体です。
        問2反応後、しばらくすると風船がしぼむのはなぜか。
        正解はイ。アンモニアは水にとけやすい気体です。容器内の水にとけることで、気体の量が減って風船がしぼみます。
        問3風船がふくらんだ状態(A)としぼんだ状態(B)を比べたとき、容器全体の質量はどうなるか。
        正解はウ。アンモニアは容器の外に出ていません。気体の状態と水にとけた状態の違いがあるだけで、原子の数は変わりません。
        問4気体が容器の外に出ていない実験では、見た目が変化しても質量は変わらない。
        ○ 正解です。原子の組み合わせや状態が変わっても、原子の数は変わらないので、質量も変わりません。
