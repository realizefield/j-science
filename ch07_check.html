確認問題:化学反応式 ★★☆

江原予備校テキストの練習問題に挑戦しよう。単元ごとに分かれているので、習った内容を確認しながら進めてください。

1化学反応式の原則と作り方

📖 江原予備校テキストより
化学反応式の基本ルールと、作り方の手順を確認します。

✏️ 練習1:反応式の原則
問1化学反応式の左辺と右辺について、正しい説明はどれか。
正解はイ。化学反応の前後では原子の組み合わせが変わるだけで、原子の種類と個数は変化しないのが原則です。
問2「2H₂O」の前にある「2」は、水分子(H₂O)が2個あることを表している。
○ 正解です。化学式の前にある数字(係数)は分子の個数を表します。中の小さい数字(添字)は1分子内の原子の数です。
問3銅の酸化「2Cu + O₂ → 2CuO」で、左辺に「2Cu」と書かれている理由として正しいものはどれか。
正解はイ。酸素はO₂(分子)で存在するので右辺のCuOは2個必要。それに合わせて左辺の銅も2個必要になります。「左辺と右辺で各原子の数を一致させる」のが係数の役割です。
2酸化・燃焼の反応式

📖 江原予備校テキスト 練習12より
酸化と燃焼に関する重要な化学反応式を確認します。

✏️ 練習2:酸化・燃焼
問1銅の酸化を表す化学反応式として正しいものはどれか。
正解はウ:2Cu + O₂ → 2CuO。生成物の酸化銅は黒色です。1000℃以下でゆるやかに酸化して黒色のCuOができます。
問2マグネシウムの燃焼の説明として正しいものはどれか。
正解はイ。2Mg + O₂ → 2MgO が正しい反応式。生成する酸化マグネシウムは白色。激しい光と熱を出す酸化なので燃焼と呼ばれます。
問3銀の酸化の化学反応式として正しいものはどれか。

正解:ウ:4Ag + O₂ → 2Ag₂O

酸化銀は Ag₂O(銀2個に酸素1個)。係数を合わせると:

  • 左辺:Ag 4個、O 2個
  • 右辺:Ag 2×2=4個、O 2個 → 一致!
問4水素の燃焼の化学反応式として正しいものはどれか。
正解はイ:2H₂ + O₂ → 2H₂O。水素も酸素も気体は分子(H₂、O₂)で存在します。水素は燃やすと水ができます。
3分解の反応式

📖 江原予備校テキスト 練習12より
第1講で学んだ「分解」を化学反応式で表します。

✏️ 練習3:分解の反応式
問1酸化銀の加熱分解の化学反応式として正しいものはどれか。
正解はイ:2Ag₂O → 4Ag + O₂。酸素は気体として分子(O₂)になるので、左辺をAg₂O 2個にして調整します。黒色の酸化銀から白色(磨くと銀色)の銀と酸素ができます。
問2水の電気分解の化学反応式として正しいものはどれか。

正解:ウ:2H₂O → 2H₂ + O₂

この式から、水素と酸素の発生体積比が分かります:

  • 水素(−極):酸素(+極) = 2:1
  • 暗記法「マスイ(麻酔)」= マイナス・水素の意味
問3炭酸水素ナトリウムの加熱分解の化学反応式として正しいものはどれか。
正解はイ:2NaHCO₃ → Na₂CO₃ + CO₂ + H₂O。3つの物質ができる重要な反応です。
暗記法:「2人の ナホコ(NaHCO₃) さんは ナツコ(Na₂CO₃) さんにかわる」+ CO₂(二酸化炭素)+ H₂O(水)
問4分解の化学反応式では、左辺の物質1種類が、右辺で2種類以上の物質に分かれる。
○ 正解です。分解は1種類の物質が2種類以上に分かれる化学変化。たとえば 2NaHCO₃ → Na₂CO₃ + CO₂ + H₂O では、左辺1種類が右辺3種類に分かれています。
4化合・還元の反応式

📖 江原予備校テキスト 練習12より
硫黄との化合や、酸化銅の還元を化学反応式で表します。

✏️ 練習4:化合・還元の反応式
問1鉄と硫黄の加熱(化合)の化学反応式として正しいものはどれか。
正解はア:Fe + S → FeS。鉄(Fe)も硫黄(S)も原子のまま反応するので係数は1。生成する硫化鉄(FeS)は黒色。
※もとの鉄と違い、磁石につかない、塩酸を加えると硫化水素(腐卵臭)が発生します。
問2酸化銅と炭素の反応(還元)の化学反応式として正しいものはどれか。

正解:ウ:2CuO + C → 2Cu + CO₂

原子の数を合わせると:

  • 左辺:Cu 2個、O 2個、C 1個
  • 右辺:Cu 2個、O 2個、C 1個 → 一致!
問3酸化銅と水素の反応(還元)の化学反応式として正しいものはどれか。
正解はイ:CuO + H₂ → Cu + H₂O。Cu 1個、O 1個、H 2個 ずつでバランスがとれているので係数は不要(1)です。
問4「2CuO + C → 2Cu + CO₂」では、酸化銅が還元され、同時に炭素が酸化されている。
○ 正解です。還元と酸化は同時に起こるのが鉄則。CuOから酸素がうばわれてCu(還元)、Cが酸素と結びついてCO₂(酸化)になっています。
