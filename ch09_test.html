1大問1:マグネシウムの酸化と質量比 〈新潟県〉

🏆 新潟県 公立高校入試問題
質量の異なるマグネシウムを、それぞれ空気中で加熱して完全に酸化させ、できた酸化マグネシウムの質量を測定したところ、次の表の結果を得た。

マグネシウムの質量(g) 0.3 0.6 0.9 1.2
酸化マグネシウムの質量(g) 0.5 1.0 1.5 2.0
✏️ 大問1(全3問)
問1表をもとにすると、酸化マグネシウムに含まれるマグネシウムの質量と酸素の質量の比はどうなるか。

正解:ウ:3 : 2

表から例えばMg 0.3g→MgO 0.5g なので:

  • 結びついた酸素 = 0.5 − 0.3 = 0.2g
  • Mg : O₂ = 0.3 : 0.2 = 3 : 2
  • 暗記法「マーさんは同士」 → Mg 3、O₂ 2
問2マグネシウムの酸化の化学反応式として正しいものはどれか。
正解はイ:2Mg + O₂ → 2MgO。左右の原子数を合わせるための係数2。
左辺:Mg 2個、O 2個 / 右辺:Mg 2個、O 2個 で一致。
問3マグネシウム 1.8g を空気中で加熱し、完全に酸化させる前に加熱をやめたところ、質量を測定したら 2.8g になっていた。酸化されていないマグネシウムの質量はいくらか。
(Mg : O₂ = 3 : 2)

正解:ア:0.3 g

  • ① 結びついた酸素 = 2.8 − 1.8 = 1.0g
  • ② 反応したMg(χg)を求める:Mg : O₂ = 3 : 2
    χ : 1.0 = 3 : 2 → 2χ = 3.0 → χ = 1.5g
  • ③ 酸化しなかったMg = 1.8 − 1.5 = 0.3 g
2大問2:燃え残ったマグネシウム 〈和歌山県〉

🏆 和歌山県 公立高校入試問題
1.0g のマグネシウムを燃焼させたとき、ピンセットではさんでいた部分が燃えずに残ってしまった。燃焼後の物質全体の質量をはかると 1.6g だった。
(マグネシウム:酸化マグネシウム = 3 : 5)

✏️ 大問2(全3問)
問1マグネシウムと結びつく酸素の質量比 Mg : O₂ はいくらか。

正解:イ:3 : 2

Mg : MgO = 3 : 5 から、結びつく酸素は 5 − 3 = 2 なので:

  • Mg : O₂ = 3 : (5−3) = 3 : 2
問2燃焼で結びついた酸素の質量はいくらか。

正解:エ:0.6 g

質量保存の法則により、増加した分が結びついた酸素です。

  • 結びついた酸素 = 1.6 − 1.0 = 0.6 g
問3燃えずに残ったマグネシウムの質量はいくらか。

正解:ア:0.1 g

  • ① 結びついた酸素 = 0.6g(問2より)
  • ② 反応したMg(χg)を求める:Mg : O₂ = 3 : 2
    χ : 0.6 = 3 : 2 → 2χ = 1.8 → χ = 0.9g
  • ③ 燃え残ったMg = 1.0 − 0.9 = 0.1 g
3大問3:銅の途中酸化 〈石川県〉

🏆 石川県 公立高校入試問題
銅の粉末をステンレス皿にとり、加熱して完全に酸化させた実験で、銅 0.4g から酸化銅 0.5g ができた。次に、銅 0.6g を加熱し、実験の途中で質量を測定したところ、質量が 0.1g 増加していた。

✏️ 大問3(全3問)
問1銅と酸素の質量比 Cu : O₂ はいくらか。

正解:イ:4 : 1

銅 0.4g から酸化銅 0.5g ができたので:

  • 結びついた酸素 = 0.5 − 0.4 = 0.1g
  • Cu : O₂ = 0.4 : 0.1 = 4 : 1
問2銅 0.6g を加熱し質量が 0.1g 増加したとき、酸化された銅の質量はいくらか。

正解:ウ:0.4 g

増加した質量 = 結びついた酸素 = 0.1g。Cu : O₂ = 4 : 1 を使います:

  • χ : 0.1 = 4 : 1
  • χ = 0.1 × 4 = 0.4 g
問3このとき、まだ反応していない銅は何 g か。

正解:イ:0.2 g

  • 反応していない銅 = 元の銅 − 反応した銅
  • = 0.6 − 0.4 = 0.2 g
4大問4:銅とマグネシウムの比較 〈山形県〉

🏆 山形県 公立高校入試問題
銅の粉末・マグネシウムの粉末をそれぞれ加熱して完全に酸化させ、できた酸化物の質量を測定したところ、次の結果を得た。

銅の粉末の質量(g) 0.60 1.20 1.80
得られた酸化銅の質量(g) 0.75 1.50 2.25
マグネシウムの粉末の質量(g) 0.60 1.20 1.80
得られた酸化マグネシウムの質量(g) 1.00 2.00 3.00
✏️ 大問4(全3問)
問1銅 1.20g と結びつく酸素の質量はいくらか。

正解:イ:0.30 g

  • 結びついた酸素 = 1.50 − 1.20 = 0.30 g
問2同じ質量の銅とマグネシウムを完全に酸化させたとき、銅に結びつく酸素の質量とマグネシウムに結びつく酸素の質量の比はいくらか。

正解:ア:3 : 8

銅とマグネシウムを同じ 1.20g にしたときの結びつく酸素を比較します。

  • 銅と結びつく酸素 = 1.50 − 1.20 = 0.30g
  • Mgと結びつく酸素 = 2.00 − 1.20 = 0.80g
  • 比 = 0.30 : 0.80 = 3 : 8
問3銅の粉末 2.20g を完全に酸化させたとき、得られる酸化銅の質量はいくらか。

正解:エ:2.75 g

銅:酸化銅の比は 1.20 : 1.50 = 4 : 5。求める酸化銅を χg とすると:

  • 4 : 5 = 2.20 : χ
  • 4 × χ = 5 × 2.20
  • 4χ = 11
  • χ = 11 ÷ 4 = 2.75 g
5大問5:化学変化と質量計算の総合問題

🏆 総合問題
第9講で学んだ「質量比を使った計算」のまとめテストです。

✏️ 大問5(全4問)
問1銅 8.0g を完全に酸化させると、できる酸化銅の質量はいくらか。
(Cu : CuO = 4 : 5)

正解:ウ:10.0 g

  • 4 : 5 = 8.0 : χ
  • 4χ = 5 × 8.0 = 40
  • χ = 40 ÷ 4 = 10.0 g
問2マグネシウム 4.5g を完全に酸化させると、結びつく酸素の質量はいくらか。
(Mg : O₂ = 3 : 2)

正解:イ:3.0 g

  • 3 : 2 = 4.5 : χ
  • 3χ = 2 × 4.5 = 9
  • χ = 9 ÷ 3 = 3.0 g
問3銅 2.0g を加熱したら、全体が 2.4g になった。酸化しなかった銅は何 g か。
(Cu : O₂ = 4 : 1)

正解:イ:0.4 g

  • ① 結びついた酸素 = 2.4 − 2.0 = 0.4g
  • ② 反応した銅 = 0.4 × 4 = 1.6g
  • ③ 酸化しなかった銅 = 2.0 − 1.6 = 0.4 g
問4金属を加熱して酸化させたとき、結びつく酸素の質量は、もとの金属の質量に比例する。
○ 正解です。Cu : O₂ = 4 : 1、Mg : O₂ = 3 : 2 のように、金属と酸素は決まった質量比で結びつきます。だから金属の量を増やせば結びつく酸素も比例して増えます(原点を通る直線のグラフになります)。
