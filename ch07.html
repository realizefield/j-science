第7講 / 化学変化を式で表す

化学反応式 ★★★

第6講で学んだ元素記号・化学式・原子の数え方を使って、いよいよ化学反応式を作ります。これまで学んだすべての反応を、ひとつの式で表せるようになりましょう。

1化学反応式の原則 ― 左辺と右辺はつり合う
📖 説明
0:00.0

これまでの講で、いろいろな化学変化を学んできました。たとえば、銅を加熱すると酸素と結びついて酸化銅ができる、という反応です。これを言葉ではなく記号で表したものを「化学反応式」といいます。

化学反応式のいちばん大事なルールは、次のとおりです。

⚗️ 化学反応式の原則

反応の前と後で、それぞれの原子の数は変わらない。

→ 化学反応式の 左辺(反応前) と 右辺(反応後) で、すべての原子の数が等しくなる。

なぜでしょうか?化学変化は、原子の組み合わせが変わるだけで、原子そのものが新しく生まれたり消えたりすることはありません。たとえば「Cu原子が左に2個あったら、右にも必ず2個ある」ということです。

▶ 例:銅の酸化

銅(Cu)が酸素(O₂)と結びついて、酸化銅(CuO)ができる反応を考えます。これを完成した化学反応式で書くと、次のようになります。

2Cu + O₂ → 2CuO

左辺と右辺で、それぞれの原子の数を数えてみましょう。

  • Cu原子 …… 左:2個、右:2個 ✓
  • O原子 …… 左:2個、右:2個 ✓

このように、左辺と右辺の原子の数がぴったり等しくなるように調整するのが化学反応式の作り方です。次の単元で、その作り方を練習しましょう。

★ 覚えよう!
  • 化学反応式 …… 化学変化を記号で表したもの
  • 大原則:左辺と右辺で、それぞれの原子の数は等しい
  • 理由:原子の組み合わせが変わるだけで、原子の数は変わらない
  • 「→」の左が反応前、右が反応後

✏️ ミニテスト(全4問)
問1化学反応式では、左辺と右辺で原子の数が等しくなる。
○ 正解です。これが化学反応式の最も大事な原則です。
問2化学変化では、原子そのものが新しく生まれたり消えたりすることがある。
× 化学変化では、原子の組み合わせが変わるだけです。原子そのものが生まれたり消えたりはしません。
問3「2Cu + O₂ → 2CuO」の左辺にあるCu原子の数はいくつか。
正解はイ:2個。「2Cu」は Cu 原子が2個あることを意味します。
問4「2H₂O」に含まれるO原子の数はいくつか。
正解はウ:2個。水分子が2個、各分子にOが1個ずつなので 2×1 = 2個です。

    2化学反応式の作り方 ― 銅の酸化を例に
    📖 説明
    0:00.0

    では、実際に化学反応式を作る練習をしましょう。銅の酸化を例にとって、ステップごとに進めていきます。

    銅(Cu)を空気中で加熱すると、酸素(O₂)と結びついて、酸化銅(CuO)ができます。これを化学反応式で表してみましょう。

    ▶ 4つのステップ

    ▶ 作り方のコツ

    化学反応式を作るとき、気をつけるポイントがあります。

    ⚗️ 大事な4つのコツ
    1. 化学式の中の小さな数字(右下)は変えない。CuO を C₂uO とか CuO₂ にしてはいけません。
    2. 調整できるのは 化学式の前の大きな数字(係数)だけ。例:Cu → 2Cu のように変える。
    3. 左辺と右辺で、登場するすべての原子の数を等しくする。
    4. 係数は、できるだけ簡単な整数にそろえる(2:4ではなく1:2のように)。

    係数1のときは「1」を書かずに省略します(「1Cu」ではなく「Cu」と書く)。

    ★ 覚えよう!
    • Step1:化学式をならべる(矢印の左に反応前、右に反応後)
    • Step2〜3:左辺=右辺になるように、化学式の前の数字(係数)を調整する
    • 変えてよいのは「化学式の前の大きな数字」だけ
    • 「中の小さな数字」は絶対に変えない(化学式そのものが変わってしまう)
    • 係数は簡単な整数にそろえる(係数1は省略)
    📚 語群 ― タップして選び、説明文の空欄をタップ!
    ✏️ ミニテスト(全4問)
    問1化学反応式を作るとき、調整してよいのはどれか。
    正解はイ。化学式の前の係数だけを調整します。中の小さな数字を変えると別の物質になってしまいます。
    問2「Cu + O₂ → CuO」を正しい化学反応式にしたものはどれか。
    正解はウ。アとイは化学式の中身を変えてしまっているので不正解。エは元の式を変えています。
    問3「2H₂ + O₂ → 2H₂O」で、左辺と右辺のH原子の数はそれぞれいくつか。
    正解はエ。左:2H₂ → H 2×2=4個、右:2H₂O → H 2×2=4個。
    問4化学反応式の係数が「1」のとき、「1」と書かないといけない。
    × 係数1は省略します。「1Cu」ではなく「Cu」と書きます。

      3覚えるべき化学反応式 ― 9個まとめて確認しよう
      📖 説明
      0:00.0

      これまでの講で学んだ化学変化を、すべて化学反応式で書いてみましょう。中学理科で必ず覚えるべきものは、次の9個です。

      ▶ 酸化(化合)の反応 ― 4つ

      1. 銅の酸化:2Cu + O₂ → 2CuO(酸化銅、黒色)
      2. マグネシウムの燃焼:2Mg + O₂ → 2MgO(酸化マグネシウム、白色)
      3. 水素の燃焼:2H₂ + O₂ → 2H₂O(水)
      4. 鉄と硫黄の化合:Fe + S → FeS(硫化鉄、黒色)

      ▶ 分解の反応 ― 3つ

      1. 酸化銀の加熱分解:2Ag₂O → 4Ag + O₂
      2. 水の電気分解:2H₂O → 2H₂ + O₂
      3. 炭酸水素ナトリウムの加熱分解:2NaHCO₃ → Na₂CO₃ + CO₂ + H₂O
      💡 炭酸水素ナトリウムの分解の覚え方(ゴロ合わせ)

      「2人のナホコさんは、ナツコさんにかわる」

      2人の
      ナホコさん      		 → ナツコさん
      にかわる
      2 NaHCO₃ → Na₂CO₃ + CO₂ + H₂O
      「ナ・ホ・コ」=
      Na・H・CO₃      		 「ナ・ツ(2)・コ」=
      Na₂・CO₃
      (残りはCO₂とH₂O)

      分解の前後で、Na・H・C・O の原子の数がぴったり等しくなっていることも確認しましょう。

      ▶ 還元の反応 ― 2つ

      1. 酸化銅+炭素:2CuO + C → 2Cu + CO₂
      2. 酸化銅+水素:CuO + H₂ → Cu + H₂O

      9個の式は、すぐには全部覚えられないかもしれません。何度も見て、書いて、声に出して、少しずつ身につけていきましょう。次の単元の練習問題でも何度も登場します。

      ★ 覚えよう!
      • 酸化(化合)4個:銅・マグネシウム・水素・鉄+硫黄
      • 分解3個:酸化銀・水・炭酸水素ナトリウム
      • 還元2個:酸化銅+炭素・酸化銅+水素
      • 暗記法:「2人のナホコさんはナツコさんにかわる」(NaHCO₃の分解)
      📚 語群 ― タップして選び、説明文の空欄をタップ!
      ✏️ ミニテスト(全6問)
      問1銅の酸化の化学反応式として正しいものはどれか。
      正解はウ。酸素はO₂分子で存在し、左右の原子数を合わせると2Cu+O₂→2CuOになります。
      問2水素の燃焼の化学反応式として正しいものはどれか。
      正解はイ。水素分子(H₂)2個と酸素分子(O₂)1個から、水分子(H₂O)2個ができます。
      問3水の電気分解の化学反応式として正しいものはどれか。
      正解はア。水素と酸素は分子(H₂、O₂)として発生します。陰極にH₂、陽極にO₂が体積比2:1で出ます。
      問4炭酸水素ナトリウムの加熱分解の化学反応式として正しいものはどれか。
      正解はエ。「2人のナホコさんはナツコさんにかわる」がポイント。左の係数2を忘れないように!
      問5酸化銅と炭素の還元反応の化学反応式として正しいものはどれか。
      正解はウ。CO₂のO原子は2個なので、左辺もO原子を2個にするためCuOの前に2をつけます。
      問6鉄と硫黄の化合の化学反応式は「Fe + S → FeS」で、係数はどれも1である。
      ○ 正解です。Fe原子1個とS原子1個が結びついてFeS(硫化鉄)になります。係数はすべて1なので省略されています。
