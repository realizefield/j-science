第5講 / 酸化と反対の変化

還元 ★★☆

酸化銅から酸素をとりさる「還元」を学びます。酸化と還元はいつも同時に起こります。炭素を使う方法と、水素を使う方法、2つの実験を見ていきましょう。

1還元って何?酸化との関係は?
📖 説明
0:00.0

第4講で「酸化」を学びました。酸化は、物質が酸素と結びつく化学変化のことでしたね。たとえば、銅(Cu)を加熱すると、空気中の酸素と結びついて酸化銅(CuO)ができました。

今回学ぶ「還元」は、酸化とは反対の変化です。つまり、酸化物から酸素をとりさる化学変化のことを「還元」といいます。

たとえば、酸化銅(CuO)から酸素を取り去ると、もとの銅(Cu)にもどります。これが還元です。

ここで大事なポイントです。酸化と還元は、いつも同時に起こります。なぜでしょうか?酸化銅から酸素を取り去るには、その酸素を「もらってくれる物質」が必要だからです。たとえば炭素や水素が、酸化銅から酸素をうばって自分自身が酸化されます。

つまり、ある物質が還元されるとき、別の物質が酸化される。これが還元の基本のしくみです。

★ 覚えよう!
  • 酸化 …… 物質が酸素と結びつく化学変化
  • 還元 …… 酸化物から酸素をとりさる化学変化
  • 酸化と還元は、いつも同時に起こる
  • 酸化銅(CuO)+ 酸素を取る物質 → 銅(Cu)+ 別の物質

✏️ ミニテスト(全3問)
問1還元とは、酸化物から酸素をとりさる化学変化のことである。
○ 正解です。「還(かえ)る・元(もと)」と書いて、酸化物から酸素を取って「もとの物質にもどす」ことを意味します。
問2酸化と還元は、いつも同時に起こる。
○ 正解です。ある物質が還元される(酸素を失う)とき、その酸素はかならず別の物質に渡されて、その物質が酸化されます。
問3次のうち、「還元」にあたる変化はどれか。
正解はイ。酸化銅(CuO)から酸素がとれて銅(Cu)にもどる変化が還元です。アとエは酸化、ウは分解です。

🏆 ベストタイム

    2酸化銅と炭素を加熱してみよう
    📖 説明
    0:00.0

    黒色の酸化銅の粉末と、炭素の粉末をよく混ぜて、試験管に入れて加熱します。下の図のように、発生する気体は石灰水を入れた試験管に通します。

    ▲ 酸化銅と炭素を試験管Aで加熱し、発生した気体を試験管Bの石灰水に通す

    ▶ 実験で起こる変化

    加熱すると、酸化銅の中の酸素と炭素が結びつき、気体の二酸化炭素が発生します。この気体を石灰水に通すと、石灰水は白くにごります。これが二酸化炭素が発生したという証拠です。

    試験管Aの中身も変化します。黒色だった混合物の中で、酸化銅は酸素をうばわれて赤褐色の銅(Cu)になります。実験のあと、試験管の中身を取り出して水で洗うと、底に赤褐色の銅が残っているのが見えます。

    ▶ 反応のしくみ

    このとき、2つの変化が同時に起こっています。

    • 酸化銅(CuO)は酸素をうばわれて、銅(Cu)になる → 還元された
    • 炭素(C)は酸素と結びついて、二酸化炭素(CO₂)になる → 酸化された
    ⚗️ 化学反応式

    2CuO + C → 2Cu + CO₂

    酸化銅(黒) + 炭素 → 銅(赤褐色) + 二酸化炭素

    ▶ 実験のとき気をつけること

    実験を終わるときは、火を消す前にガラス管を石灰水から取り出します。なぜでしょうか?もし先に火を消してしまうと、試験管Aの中の空気がひえて、石灰水が試験管Aに逆流してしまうからです。冷たい液体が熱い試験管に入ると、試験管が割れてしまうおそれがあります。

    ★ 覚えよう!
    • 酸化銅(黒)+ 炭素 → 銅(赤褐色)+ 二酸化炭素
    • 化学反応式:2CuO + C → 2Cu + CO₂
    • 石灰水が白くにごる → 二酸化炭素が発生した証拠
    • 試験管の口が黒から赤褐色に変化 → 銅ができた証拠
    • 火を消す前にガラス管を石灰水から取り出す(逆流防止)
    📚 語群 ― タップして選び、説明文の空欄をタップ!
    ✏️ ミニテスト(全4問)
    問1酸化銅と炭素を加熱すると、銅と二酸化炭素ができる。
    ○ 正解です。2CuO + C → 2Cu + CO₂ の反応です。
    問2酸化銅と炭素を加熱した実験で、石灰水を白くにごらせた気体は何か。
    正解はイ:二酸化炭素。石灰水が白くにごる気体は、二酸化炭素の特徴です。
    問3酸化銅と炭素を加熱した実験で、酸化銅と炭素はそれぞれ何になったか。
    正解はウ。酸化銅は酸素をうばわれて還元、炭素は酸素と結びついて酸化されます。
    問4酸化銅と炭素を加熱する実験で、火を消す前にガラス管を石灰水から取り出す理由はどれか。
    正解はア。先に火を消すと、試験管Aの中の気体が冷えて体積が小さくなり、石灰水が逆流します。

      3酸化銅を水素で還元してみよう
      📖 説明
      0:00.0

      還元には、炭素を使う方法のほかに、水素を使う方法もあります。下の図のように、横向きにした試験管の中に酸化銅を入れて、左から水素を流しながら加熱します。

      ▲ 横向きの試験管の中央に酸化銅を入れ、水素を流しながら加熱する

      ▶ 実験で起こる変化

      加熱すると、酸化銅の中の酸素と水素が結びついて、水(H₂O)ができます。試験管の中をよく見ると、加熱したところより少し離れた場所(出口側)に水滴がついているのが見えます。

      そして酸化銅は、酸素をうばわれて赤褐色の銅になります。黒色から赤褐色に色が変わるので、見ただけで還元が起きたとわかります。

      ▶ 反応のしくみ

      この実験でも、酸化と還元が同時に起こっています。

      • 酸化銅(CuO)は酸素をうばわれて、銅(Cu)になる → 還元された
      • 水素(H₂)は酸素と結びついて、水(H₂O)になる → 酸化された
      ⚗️ 化学反応式

      CuO + H₂ → Cu + H₂O

      酸化銅(黒) + 水素 → 銅(赤褐色) + 水

      ▶ 質量の変化

      酸化銅から酸素が取れて銅になるので、加熱前と加熱後で質量を比べると、加熱後のほうが軽くなります(酸素の分だけ減る)。これも還元が起きた証拠の一つです。

      ★ 覚えよう!
      • 酸化銅(黒)+ 水素 → 銅(赤褐色)+ 水
      • 化学反応式:CuO + H₂ → Cu + H₂O
      • 試験管の出口側に水滴ができる
      • 酸化銅の色が黒から赤褐色に変わる
      • 加熱後は酸素の分だけ質量が減る(軽くなる)
      📚 語群 ― タップして選び、説明文の空欄をタップ!
      ✏️ ミニテスト(全4問)
      問1酸化銅と水素を反応させると、銅と水ができる。
      ○ 正解です。CuO + H₂ → Cu + H₂O の反応です。
      問2酸化銅を水素で還元するとき、酸化銅の色は何色から何色に変わるか。
      正解はア。黒色の酸化銅が、赤褐色の銅にもどります。
      問3酸化銅と水素を反応させたとき、加熱後の質量は加熱前と比べてどうなるか。
      正解はイ。酸化銅から酸素が取れて銅になるので、加熱前の酸化銅のほうが酸素の分だけ重いです。
      問4酸化銅+水素の還元実験で、試験管の出口付近にできるものは何か。
      正解はエ:水滴。水素+酸素 → 水 ができるので、出口付近に水滴がつきます。
