練習問題:化学変化と質量計算 ★☆☆

レッスンで学んだ「質量比を使った計算」の確認です。それぞれ4つの選択肢から正しいものを1つタップしてください。すぐに答え合わせがされます。

✏️ 練習問題(全10問)
問1銅と酸素が結びついて酸化銅ができるとき、銅と酸素の質量比として正しいものはどれか。
正解はイ:Cu : O₂ = 4 : 1。銅:酸素:酸化銅 = 4 : 1 : 5 が基本。語呂「マーさんは同士」のCu(銅)4を覚えておきましょう。
問2マグネシウムと酸素が結びついて酸化マグネシウムができるとき、Mg : O₂ : MgO の質量比として正しいものはどれか。
正解はア:3 : 2 : 5。語呂「マーさんは同士」のMg(マグネシウム)3と酸素2。
左辺(Mg+O₂)= 3 + 2 = 5、右辺(MgO)= 5 で一致しています。
問3銅 1.6g を完全に酸化させると、結びつく酸素の質量はいくらか。
(Cu : O₂ = 4 : 1)

正解:ウ:0.4 g

Cu : O₂ = 4 : 1 を使います。結びつく酸素を χg とすると:

  • 4 : 1 = 1.6 : χ
  • 4 × χ = 1 × 1.6
  • 4χ = 1.6
  • χ = 1.6 ÷ 4 = 0.4 g
問4銅 2.4g を完全に酸化させると、できる酸化銅の質量はいくらか。
(Cu : CuO = 4 : 5)

正解:ウ:3.0 g

Cu : CuO = 4 : 5 を使います。できる酸化銅を χg とすると:

  • 4 : 5 = 2.4 : χ
  • 4 × χ = 5 × 2.4
  • 4χ = 12
  • χ = 12 ÷ 4 = 3.0 g
問5マグネシウム 0.6g を完全に酸化させると、結びつく酸素の質量はいくらか。
(Mg : O₂ = 3 : 2)

正解:イ:0.4 g

Mg : O₂ = 3 : 2 を使います。結びつく酸素を χg とすると:

  • 3 : 2 = 0.6 : χ
  • 3 × χ = 2 × 0.6
  • 3χ = 1.2
  • χ = 1.2 ÷ 3 = 0.4 g
問6マグネシウム 1.5g を完全に酸化させると、できる酸化マグネシウムの質量はいくらか。
(Mg : MgO = 3 : 5)

正解:ウ:2.5 g

Mg : MgO = 3 : 5 を使います。できる酸化マグネシウムを χg とすると:

  • 3 : 5 = 1.5 : χ
  • 3 × χ = 5 × 1.5
  • 3χ = 7.5
  • χ = 7.5 ÷ 3 = 2.5 g
問7酸化銅 5.0g を作りたい。必要な銅の質量はいくらか。
(Cu : CuO = 4 : 5)

正解:イ:4.0 g

Cu : CuO = 4 : 5 を使います。必要な銅を χg とすると:

  • 4 : 5 = χ : 5.0
  • 5 × χ = 4 × 5.0
  • 5χ = 20
  • χ = 20 ÷ 5 = 4.0 g
問8酸化マグネシウム 4.5g を作りたい。必要なマグネシウムの質量はいくらか。
(Mg : MgO = 3 : 5)

正解:イ:2.7 g

Mg : MgO = 3 : 5 を使います。必要なMgを χg とすると:

  • 3 : 5 = χ : 4.5
  • 5 × χ = 3 × 4.5
  • 5χ = 13.5
  • χ = 13.5 ÷ 5 = 2.7 g
問9銅 1.2g を加熱したところ、かき混ぜが不十分で全体が 1.4g にしかならなかった。酸化しなかった銅は何 g か。
(Cu : O₂ = 4 : 1)

正解:イ:0.4 g

3ステップで解きます:

  • ① 結びついた酸素 = 1.4 − 1.2 = 0.2g
  • ② 反応した銅(χg)を求める:Cu : O₂ = 4 : 1 より χ : 0.2 = 4 : 1
    χ = 0.2 × 4 = 0.8g
  • ③ 酸化しなかった銅 = 1.2 − 0.8 = 0.4 g
問10マグネシウム 1.0g を加熱したところ、酸化が不十分で全体が 1.6g になった。酸化しなかったマグネシウムは何 g か。
(Mg : O₂ = 3 : 2)

正解:イ:0.1 g

3ステップで解きます:

  • ① 結びついた酸素 = 1.6 − 1.0 = 0.6g
  • ② 反応したMg(χg)を求める:Mg : O₂ = 3 : 2 より χ : 0.6 = 3 : 2
    2χ = 3 × 0.6 = 1.8 → χ = 0.9g
  • ③ 酸化しなかったMg = 1.0 − 0.9 = 0.1 g
