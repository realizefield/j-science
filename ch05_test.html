単元末テスト:還元 ★★★

入試レベルの問題に挑戦!各都道府県の高校入試で実際に出題された問題から構成しています。じっくり考えて解いてみましょう。

1大問1:還元と酸化の関係 〈愛媛県改〉

🏆 愛媛県 公立高校入試問題(改題)
酸化銅と炭素の粉末を混ぜ、加熱する実験を行った。この実験で起こった化学変化について答えなさい。

✏️ 大問1(全4問)
問1反応のうち、酸化銅から銅への変化(P)は何か。
正解はイ:還元。酸化銅(CuO)から酸素がうばわれて銅(Cu)になっているので還元です。
問2反応のうち、炭素から二酸化炭素への変化(Q)は何か。
正解はア:酸化。炭素(C)が酸素と結びついて二酸化炭素(CO₂)になっているので酸化です。
問3発生した気体A(石灰水を白くにごらせる)の化学式は何か。
正解はウ:CO₂(二酸化炭素)。石灰水を白くにごらせる気体は二酸化炭素です。
問4反応後に得られた物質B(赤褐色)の化学式は何か。
正解はイ:Cu(銅)。赤褐色の金属で、電気をよく通します。
2大問2:還元される物質はどれ? 〈三重県〉

🏆 三重県 公立高校入試問題
酸化銅と炭素粉末を試験管Aに入れて加熱する実験を行った。

✏️ 大問2(全3問)
問1この実験での酸化銅と炭素粉末の変化として正しいものはどれか。
正解はウ。これは入試で頻出の組み合わせ問題です。酸化銅(CuO)→銅(Cu) は酸素をうばわれた = 還元、炭素(C)→二酸化炭素(CO₂) は酸素と結びついた = 酸化。
問2反応後にできた銅の性質を確かめる方法として正しいものはどれか。
正解はイ。銅は金属なので、こすると金属光沢が出ます。電気を通すか、たたくとよくのびるかも金属の特徴です。磁石にくっつくのは鉄やニッケルなど一部の金属だけ。
問3この実験で起きた化学変化を化学反応式で表すとどうなるか。
正解はイ:2CuO + C → 2Cu + CO₂。原子の数を合わせる(係数を調整する)ことが重要です。Cu 2個・O 2個・C 1個 が左右で一致しています。
3大問3:質量計算問題 〈山梨県〉

🏆 山梨県 公立高校入試問題
酸化銅 4.00g と炭素 0.30g を混ぜて加熱したところ、加熱後の固体の質量は 3.73g になった。

✏️ 大問3(全3問)
問1この反応で、酸化された物質と還元された物質の組み合わせとして正しいものはどれか。
正解はイ。炭素は酸素と結びついて二酸化炭素になる(酸化)、酸化銅は酸素をうばわれて銅になる(還元)。
問2反応の前後で、減った質量(発生した二酸化炭素の質量)はいくらか。
(酸化銅 4.00g + 炭素 0.30g = 反応前 4.30g、反応後の固体 3.73g)

正解:イ:0.57 g

質量保存の法則を使います。発生した二酸化炭素は容器の外へ出るので、その分だけ固体の質量は減ります。

  • 反応前の全体 = 4.00 + 0.30 = 4.30g
  • 反応後の固体 = 3.73g
  • 発生した二酸化炭素 = 4.30 − 3.73 = 0.57g
問3加熱後の固体 3.73g 中に含まれる単体の銅は何 g か。
(Cu : O₂ : CuO = 4 : 1 : 5)

正解:イ:3.20 g

酸化銅 4.00g がすべて還元されたとして計算します。

  • Cu : CuO = 4 : 5 を使う。求める銅の質量を χg とすると
  • 4 : 5 = χ : 4.00
  • 5χ = 4 × 4.00 = 16
  • χ = 16 ÷ 5 = 3.20 g
4大問4:酸化銅と炭素の量の関係 〈大分県〉

🏆 大分県 公立高校入試問題
酸化銅 4.00g に混ぜる炭素の質量を変えて加熱する実験を行った。0.15g の炭素を混ぜたとき、試験管に残った固体の質量は 3.60g だった。

✏️ 大問4(全3問)
問1炭素 0.15g がすべて反応したとすると、結びついた酸素の質量はいくらか。
(C : O₂ : CO₂ = 12 : 32 : 44 = 3 : 8 : 11)

正解:ウ:0.40 g

C : O₂ = 3 : 8 を使います。結びついた酸素を χg とすると:

  • 3 : 8 = 0.15 : χ
  • 3χ = 8 × 0.15 = 1.20
  • χ = 1.20 ÷ 3 = 0.40 g
問2問1で求めた酸素 0.40g は、もともと酸化銅から取り出されたもの。還元された酸化銅は何 g か。
(Cu : O₂ : CuO = 4 : 1 : 5)

正解:イ:2.00 g

O₂ : CuO = 1 : 5 を使います。還元された酸化銅を χg とすると:

  • 1 : 5 = 0.40 : χ
  • χ = 5 × 0.40 = 2.00 g
問3残った固体 3.60g 中に含まれる単体の銅は何 g か。
(問2で還元された酸化銅は 2.00g、その分から銅ができている)

正解:イ:1.60 g

Cu : CuO = 4 : 5 を使います。できた銅を χg とすると:

  • 4 : 5 = χ : 2.00
  • 5χ = 4 × 2.00 = 8
  • χ = 8 ÷ 5 = 1.60 g

残った固体 3.60g には、銅 1.60g + 反応しなかった酸化銅 2.00g が含まれています。

5大問5:還元の総合問題

🏆 総合問題
第5講で学んだ内容のまとめテストです。

✏️ 大問5(全4問)
問1還元反応として正しい組み合わせはどれか。
正解はイ。酸化銀(Ag₂O)から酸素がうばわれて銀(Ag)が出てくる反応は還元です。アとウは酸化、エは分解(電気分解)です。
問2製鉄で「炭素」が重要な役割を果たすのはなぜか。
正解はエ。コークス(炭素)は酸化鉄から酸素をうばう還元剤としてはたらき、鉄を取り出します。同時に炭素は酸化されて二酸化炭素になります。
問3酸化銅 8.0g を完全に還元するには、何 g の銅が得られるか。(Cu : CuO = 4 : 5)

正解:ウ:6.4 g

Cu : CuO = 4 : 5 を使います。求める銅を χg とすると:

  • 4 : 5 = χ : 8.0
  • 5χ = 4 × 8.0 = 32
  • χ = 32 ÷ 5 = 6.4 g
問4「酸化と還元はつねに同時に起こる」という関係は、すべての還元反応に当てはまる。
○ 正解です。物質Aから酸素がうばわれる(還元される)とき、その酸素は必ず別の物質Bに渡されます(酸化される)。これが化学変化の鉄則です。
