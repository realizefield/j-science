練習問題:化学反応式 ★☆☆

レッスンで学んだ「化学反応式」の確認です。それぞれ4つの選択肢から正しいものを1つタップしてください。すぐに答え合わせがされます。

✏️ 練習問題(全10問)
問1化学反応式の原則として正しいものはどれか。
正解はイ。化学反応式は左辺=右辺で各原子の数が等しいのが原則。原子は反応の前後で組み合わせが変わるだけで、数は変化しません。
問2銅の酸化の化学反応式として正しいものはどれか。

正解:ウ:2Cu + O₂ → 2CuO

酸素は分子(O₂)で存在するため、左右の原子数を合わせる必要があります。

  • 左辺:Cu 2個、O 2個
  • 右辺:Cu 2個、O 2個 → 一致!
問3マグネシウムの燃焼の化学反応式として正しいものはどれか。
正解はウ:2Mg + O₂ → 2MgO。銅の酸化と同じ形。マグネシウムも酸素もそれぞれ2個ずつで一致しています。光と熱を出す激しい酸化なので燃焼と呼ばれます。
問4水素の燃焼の化学反応式として正しいものはどれか。

正解:イ:2H₂ + O₂ → 2H₂O

水素も酸素も気体として分子(H₂、O₂)で存在します。

  • 左辺:H 4個、O 2個
  • 右辺:H 4個、O 2個 → 一致!
問5鉄と硫黄の化合の化学反応式として正しいものはどれか。
正解はア:Fe + S → FeS。鉄も硫黄も分子をつくらず原子のまま反応するので、係数は不要です。FeS は硫化鉄と読みます。
問6酸化銀の加熱分解の化学反応式として正しいものはどれか。

正解:イ:2Ag₂O → 4Ag + O₂

酸素は気体になるとO₂(分子)になるので、左辺をAg₂O 2個にして調整します。

  • 左辺:Ag 4個、O 2個
  • 右辺:Ag 4個、O 2個 → 一致!
問7水の電気分解の化学反応式として正しいものはどれか。
正解はウ:2H₂O → 2H₂ + O₂。水素は−極、酸素は+極に発生します。体積比は水素:酸素=2:1で、これは反応式の係数比と一致しています。
問8炭酸水素ナトリウムの加熱分解の化学反応式として正しいものはどれか。
正解はイ:2NaHCO₃ → Na₂CO₃ + CO₂ + H₂O。語呂で覚えましょう ―「2人のナホコ(NaHCO₃)さんは ナツコ(Na₂CO₃)さん にかわる」。Naが2個必要なので左辺は 2NaHCO₃ になります。
問9酸化銅と炭素の還元反応の化学反応式として正しいものはどれか。

正解:イ:2CuO + C → 2Cu + CO₂

炭素1個に対し酸素2個が必要(CO₂)なので、酸化銅は2個必要です。

  • 左辺:Cu 2個、O 2個、C 1個
  • 右辺:Cu 2個、O 2個、C 1個 → 一致!
問10酸化銅と水素の還元反応の化学反応式として正しいものはどれか。
正解はウ:CuO + H₂ → Cu + H₂O。左辺と右辺の原子数を確認すると、Cu 1個、O 1個、H 2個 ずつで完全にバランスがとれています。係数は1なので省略します。
