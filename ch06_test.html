単元末テスト:化学式と原子・分子 ★★★

入試レベルの問題に挑戦!化学式や原子・分子に関する入試問題を集めました。じっくり考えて解いてみましょう。

1大問1:化学式の読み解き 〈三重県改〉

🏆 三重県 公立高校入試問題(改題)
酸化銅と炭素粉末を試験管に入れて加熱したとき、発生する物質や物質の表記について答えなさい。

✏️ 大問1(全4問)
問1「酸化銅」を表す化学式として正しいものはどれか。
正解はウ:CuO。銅(Cu)1個と酸素(O)1個でできた化合物。CO は一酸化炭素、Co はコバルト。大文字小文字に注意!
問2反応で発生した気体は石灰水を白くにごらせた。この気体の化学式として正しいものはどれか。
正解はア:CO₂(二酸化炭素)。石灰水を白くにごらせる気体は二酸化炭素です。
問3反応後に試験管に残った赤色の物質を化学式で表すとどれか。
正解はイ:Cu(銅)。赤褐色の金属で、電気をよく通します。Co(コバルト)は別の元素です!
問4「2CuO」は、酸化銅の分子が2個あることを表している。
○ 正解です。化学式の前にある数字(係数)は物質(分子)の個数を表します。中の小さい数字(添字)は分子内の原子の数です。
2大問2:炭酸水素ナトリウムの化学式 〈愛媛県改〉

🏆 愛媛県 公立高校入試問題(改題)
炭酸水素ナトリウムを加熱すると、3種類の物質に分解された。それぞれの物質の化学式について答えなさい。

✏️ 大問2(全3問)
問1「炭酸水素ナトリウム」を表す化学式として正しいものはどれか。
正解はエ:NaHCO₃。「ナトリウム + 水素 + 炭素 + 酸素3」の順。Na₂CO₃ は分解後にできる炭酸ナトリウムなので注意!
問2炭酸水素ナトリウムを加熱して残った白色の固体「炭酸ナトリウム」の化学式として正しいものはどれか。
正解はイ:Na₂CO₃。ナトリウム原子が2個、炭素原子が1個、酸素原子が3個の化合物。語呂「ナホコ→ナツコ」(NaHCO₃→Na₂CO₃)で覚えましょう!
問3炭酸水素ナトリウム(NaHCO₃)1個に含まれている原子の数として正しいものはどれか。

正解:ウ:6個

NaHCO₃ には次の原子が含まれます:

  • Na(ナトリウム) 1個
  • H(水素) 1個
  • C(炭素) 1個
  • O(酸素) 3個
  • 合計 = 1 + 1 + 1 + 3 = 6個
3大問3:酸化反応と化学式 〈福島県改〉

🏆 福島県 公立高校入試問題(改題)
銅とマグネシウムをそれぞれ加熱して空気中の酸素と反応させた。生成物の化学式について答えなさい。

✏️ 大問3(全3問)
問1「酸化マグネシウム」を表す化学式として正しいものはどれか。
正解はウ:MgO。マグネシウム原子(Mg)1個と酸素原子(O)1個で結合。CuO と同じ「金属:酸素 = 1:1」のタイプ。Mn(マンガン)とMg(マグネシウム)の混同に注意!
問2「MgO」が表す物質について、正しい説明はどれか。
正解はイ。MgO は Mg(マグネシウム) と O(酸素) の2種類の元素からできているので化合物です。1種類の元素のみなら単体ですが、MgO は2種類なので化合物。化学式で表せるので混合物ではありません。
問3「2CuO」(酸化銅2個)に含まれる原子の総数として正しいものはどれか。

正解:イ:4個

CuO 1個には Cu 1個 + O 1個 = 2個の原子があります。

  • 2CuO → 2個 × 2 = 4個
4大問4:水の電気分解と化学式 〈標準入試問題〉

🏆 標準入試問題
水酸化ナトリウムをとかした水を電気分解すると、両極から気体が発生した。発生した気体の化学式について答えなさい。

✏️ 大問4(全3問)
問1−極(陰極)で発生した気体「水素」の化学式として正しいものはどれか。
正解はイ:H₂。水素は気体として存在するときは原子2個で1つの分子(H₂)を作ります。He(ヘリウム)は別の元素なので注意!
問2水分子(H₂O)1個に含まれる原子の数として正しいものはどれか。

正解:ウ:3個

H₂O には水素原子と酸素原子が次のように含まれます:

  • H(水素原子) 2個
  • O(酸素原子) 1個
  • 合計 = 2 + 1 = 3個
問3「2H₂」に含まれる水素原子は全部で何個か。

正解:エ:4個

H₂ 1個には水素原子が2個含まれます。

  • 2H₂ → 2個 × 2 = 4個
5大問5:化学式の総合問題

🏆 総合問題
第6講で学んだ「元素記号」「化学式」「物質の分類」のまとめテストです。

✏️ 大問5(全4問)
問1次の元素記号と元素名の組み合わせとして、誤っているものはどれか。
正解はイ(これが誤り)。Cℓ は塩素(chlorine)です。炭素は C(carbon)。Cℓ と C(炭素)、Cu(銅)、Co(コバルト)はすべて C で始まるので混同に注意!
問2次のうち、すべて「単体」である組み合わせはどれか。
正解はウ。H₂、O₂、N₂はすべて1種類の元素のみでできているので単体。アはH₂OとCO₂が化合物、イはNaCℓとCuOが化合物、エはNH₃が化合物・空気が混合物です。
問3「3CO₂」に含まれる原子の総数として正しいものはどれか。

正解:イ:9個

CO₂ 1個には C 1個 + O 2個 = 3個の原子。

  • 3CO₂ → 3個 × 3 = 9個
  • (炭素原子3個 + 酸素原子6個 = 9個)
問4化学式の前にある数字(係数)は、その分子1個に含まれる原子の数を表している。
× 化学式の前にある係数は物質(分子)の個数を表します。1分子に含まれる原子の数を表すのは、化学式の中の添字(小さい数字)です。例:H₂O では「H₂」が水素原子2個、「3H₂O」の3が水分子3個。
