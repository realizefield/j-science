第3講 / 酸素と結びつく化合

酸化(燃焼) ★★☆

物質が酸素と結びつく化学変化「酸化」を学びます。銅やマグネシウムを加熱すると、色が変わって新しい物質ができます。光と熱を出して激しく起こる「燃焼」もここで仲間入りです!

1酸化って何?燃焼との関係は?
📖 説明
0:00.0

第2講では「化合」を学びました。2種類以上の物質が結びついて新しい物質ができる反応でしたね。鉄+硫黄 → 硫化鉄、銅+硫黄 → 硫化銅 という例を見ました。

今回は、化合の中でも特に「酸素と結びつく化合」を学びます。物質が酸素と結びつく化学変化を「酸化」といい、酸化によってできた物質を「酸化物」とよびます。

たとえば、鉄が空気中でさびるのも酸化です。鉄がゆっくりと酸素と結びついて、酸化鉄(さび)ができています。スチールウール(鉄)を加熱したときの変化も、酸化の代表例でしたね。

そして、酸化の中でも特に光や熱を出して激しく起こる酸化を「燃焼」といいます。木が燃えるのも、ガスバーナーの火も、すべて燃焼=激しい酸化です。

つまり、関係を整理すると次のようになります(右にいくほど意味がせまい):
化学変化 ⊃ 化合 ⊃ 酸化 ⊃ 燃焼
これは「化学変化の中に化合があり、化合の中に酸化があり、酸化の中に燃焼がある」という意味です。つまり、燃焼は酸化の仲間で、酸化は化合の仲間。マトリョーシカのように入れ子になっています。

★ 覚えよう!
  • 酸化 …… 物質が酸素と結びつく化学変化(化合の一種)
  • 酸化物 …… 酸化によってできた物質
  • 燃焼 …… 光と熱を出して激しく起こる酸化
  • 関係:燃焼は酸化の仲間、酸化は化合の仲間(化合 ⊃ 酸化 ⊃ 燃焼)
⚠️ 注意
  • 「燃焼」は光と熱の両方を出すのがポイント。熱だけを出すゆっくりした酸化(さびなど)は燃焼ではない。
  • 酸化はすべて化合だが、化合が必ず酸化とはかぎらない(例:鉄+硫黄=酸素と関係ないので化合だが酸化ではない)。

✏️ ミニテスト(全4問)
問1物質が酸素と結びつく化学変化を「酸化」という。
○ 正解です。酸素と結びつく化合のことを酸化とよびます。できた物質を「酸化物」といいます。
問2燃焼とは、熱だけを出してゆっくり起こる酸化のことである。
× 燃焼は光と熱を出して激しく起こる酸化です。熱だけを出すゆっくりした酸化(さびなど)は燃焼ではありません。
問3次のうち、酸化にあたるものはどれか。
正解はイ。さびは鉄が酸素と結びつく反応なので酸化です。アは硫黄との化合なので酸化ではありません。ウは状態変化、エは溶解で化学変化ですらありません。
問4「燃焼」と「ゆるやかな酸化(さびなど)」の最大のちがいは何か。
正解はエ。どちらも酸素と結びつく酸化ですが、燃焼は光と熱を出して激しく起こる点が違います。
2銅の酸化 ― 赤色から黒色へ
📖 説明
0:00.0

銅(Cu)を空気中で加熱してみましょう。銅板や銅粉、または銅線を使います。最初は赤色(赤褐色)をしている銅が、加熱を続けるとだんだん表面が黒色に変わっていきます。

これは、銅が空気中の酸素と結びついて「酸化銅」という別の物質に変わったからです。色は黒、性質は銅とまったく違って、もろくて電気を通しません。

言葉の式で書くと:
銅 + 酸素 → 酸化銅

銅の酸化は、光や熱を出すような激しい反応ではありません。だから「燃焼」ではなく、ゆるやかな酸化です。よく見るとガスバーナーで加熱している間、銅は赤くなりますが、それはバーナーの熱によるもので、銅自身が光を出しているわけではありません。

また、酸素と結びついた分だけ質量は増えます。0.4gの銅をしっかり酸化させると、酸化銅は0.5gになります(増えた分は結びついた酸素の質量)。質量の話は第9講で計算問題としてくわしく学びます。

★ 覚えよう!
  • 銅(赤色)+ 酸素 → 酸化銅(黒色)
  • 銅の酸化はゆるやかな酸化(燃焼ではない)
  • 酸化銅は黒く、もろく、電気を通さない(銅と性質が大きく違う)
  • 結びついた酸素の分だけ質量が増える
📚 語群 ― タップして選び、説明文の空欄をタップ!
✏️ ミニテスト(全4問)
問1赤色の銅を空気中で加熱した。表面の色は何色になるか。
正解はイ:黒色。銅(赤)が酸素と結びついて酸化銅(黒)になります。色の変化が酸化の証拠です。
問2銅を加熱したときにできる物質の名前は何か。
正解はウ:酸化銅。銅+酸素 → 酸化銅。酸素と結びつくので「酸化〇〇」という名前になります。
問3銅を加熱して酸化銅にしたとき、加熱前と比べて質量はどうなるか。
正解はア。酸素が新たに結びつくので、その分だけ重くなります。化合では結びついた相手の質量だけ増えるのがポイント。
問4銅の酸化は、強い光と熱を出して激しく起こる燃焼である。
× 銅の酸化はゆるやかな酸化で、燃焼ではありません。バーナーの炎で銅が赤く見えるのは、加熱されているからで、銅自身が光を出しているわけではありません。
3マグネシウムの燃焼 ― まぶしい光と白い灰
📖 説明
0:00.0

次は、マグネシウム(Mg)というぎん色の金属を加熱してみましょう。リボン状にしたマグネシウムをピンセットでつかみ、ガスバーナーの炎に近づけます。

すると、マグネシウムは目もくらむようなまぶしい光と熱を出して、はげしく燃えます。これは、銅の酸化とちがって典型的な「燃焼」です。光と熱を出して激しく酸化する反応ですね。

燃え終わると、ぎん色のマグネシウムは消え、あとに白色の粉のようなもの(灰)が残ります。これが「酸化マグネシウム」です。

言葉の式で書くと:
マグネシウム + 酸素 → 酸化マグネシウム

銅と同じように、結びついた酸素の分だけ質量は増えます。マグネシウムの燃焼は花火や、写真のフラッシュ(昔のもの)などにも利用されてきました。

★ 覚えよう!
  • マグネシウム(ぎん色)+ 酸素 → 酸化マグネシウム(白色)
  • マグネシウムの酸化は燃焼(まぶしい光と熱を出して激しく反応)
  • 銅の酸化(ゆるやか)とくらべると、反応のはげしさが大きく違う
⚠️ 注意
  • マグネシウムの炎を直接見ると目をいためる。必ず保護メガネをかけ、長時間見続けないこと。
  • 燃焼中のマグネシウムは非常に高温なので、机の上ではなく金属皿の上で実験する。

📊 銅の酸化 vs マグネシウムの燃焼 ― 比較
項目 銅の酸化 マグネシウムの燃焼
反応のはげしさ ゆるやか 激しい(燃焼)

もとの色 赤色 ぎん色

✏️ ミニテスト(全4問)
問1マグネシウムを加熱したときの様子として正しいものはどれか。
正解はウ。マグネシウムの酸化は典型的な燃焼で、まぶしい光と熱を出して激しく起こります。
問2マグネシウムが燃えたあとに残る物質の色は?
正解はイ:白色。できる物質は酸化マグネシウムで、白色の粉(灰のように見える)です。銅の酸化(黒)としっかり区別!
問3銅の酸化とマグネシウムの燃焼の違いとして正しいものはどれか。
正解はエ。どちらも酸素と結びつく酸化ですが、反応のはげしさが大きく違います。マグネシウムの方が燃焼で、銅はゆるやかな酸化です。
問4マグネシウムが酸素と結びついて酸化マグネシウムになると、酸素の分だけ質量が増える。
○ 正解です。酸化はすべて酸素と結びつく反応なので、結びついた酸素の分だけ質量が増えます。これは銅もマグネシウムも同じです。
4銀の酸化と水素の燃焼 ― 黒くなる銀、爆発する水素
📖 説明
0:00.0

金属では、銅・マグネシウム以外にも銀(Ag)がゆっくりと酸化することが知られています。長く使った銀のスプーンやアクセサリーが、だんだん黒色くくすんでくるのを見たことはありませんか?あれは銀が空気中の酸素や硫黄と結びついて表面が変化したからです。これも酸化のひとつ(ゆるやか)。

銀の酸化でできる物質を「酸化銀」といい、これも黒色の物質です。これは第1講で学んだ「酸化銀の分解」(加熱すると銀+酸素にもどる)で出てきましたね。酸化と分解は反対の反応です。

つぎに、気体の例として水素を見てみましょう。水素を入れた試験管の口に火を近づけると、「ポンッ」という音を出して燃え、試験管の内側に小さな水のしずくがつきます。

これは水素が空気中の酸素と結びつく激しい酸化、つまり水素の燃焼です。できた物質は…なんと水!

言葉の式で書くと:
水素 + 酸素 → 水

つまり、水(H₂O)も水素という物質が酸化してできた「酸化物」のひとつなのです。少し意外ですね。

★ 覚えよう!
  • 銀(Ag)+ 酸素 → 酸化銀(黒色) ※ゆるやかな酸化
  • 水素 + 酸素 → 水 ※激しい燃焼(ポンッと音を出す)
  • 水も「水素の酸化物」ととらえることができる
  • 酸化銀の加熱(分解) ⇄ 銀の酸化 …は反対の反応

📋 酸化のまとめ ― 4つの例
物質 変化 できた物質 激しさ
銅(赤) + 酸素 → 酸化銅(黒) ゆるやか
マグネシウム(ぎん) + 酸素 → 酸化マグネシウム(白) 燃焼
銀(ぎん) + 酸素 → 酸化銀(黒) ゆるやか
水素(気体) + 酸素 → 水 燃焼(ポンッ)
✏️ ミニテスト(全4問)
問1銀のアクセサリーを長く使っていると黒くなるのはなぜか。
正解はイ。銀(ぎん色)が酸素とゆるやかに酸化して、表面が酸化銀(黒色)になります。これも酸化の一種です。
問2水素を入れた試験管の口に火をつけたとき、起きることとして正しいものはどれか。
正解はエ。水素は燃焼すると音と光を出し、酸素と結びついて水になります。試験管の内側に水滴がつくのが反応の証拠です。
問3水素 + 酸素 → ? 。空欄にあてはまるものはどれか。
正解はウ:水。水素の酸化物は水。意外ですが、ふだん飲んでいる水も「水素 + 酸素」でできています。
問4第1講で学んだ「酸化銀の加熱による分解(酸化銀 → 銀 + 酸素)」と、本講の「銀の酸化(銀 + 酸素 → 酸化銀)」は、ちょうど反対の反応である。
○ 正解です。分解と化合(酸化)はちょうど反対の反応。たとえば「酸化銀 ⇄ 銀+酸素」のように、行ったり来たりの関係です。
