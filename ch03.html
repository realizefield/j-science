化合 ★★☆

第1講では「分解」(1つの物質が2つ以上に分かれる反応)を学びました。この講で学ぶ「化合」はその逆で、2種類以上の物質が結びついて新しい1つの物質ができる反応です。鉄と硫黄、銅と硫黄の反応で実際に確かめていきましょう。

1化合 ― 物質が結びつく反応
📖 説明
0:00.0

第1講で学んだ「分解」とは、1種類の物質が2種類以上の物質に分かれる化学変化でした。今度はその 逆向きの反応 を考えます。

2種類以上の物質が結びついて、もとの物質とはちがう 新しい1つの物質 ができる化学変化を「化合」といいます。「結びつく」反応、と覚えましょう。

そして、化合によってできた新しい物質を「化合物」といいます。化合物は、もとの物質とはまったく性質がちがう、別の物質になります。

第2講で学んだ用語と組み合わせると、こんな関係になります。

● 1種類の元素からできている物質 = 単体(例: 鉄 Fe、酸素 O₂、銅 Cu)

● 2種類以上の元素からできている物質 = 化合物(例: 水 H₂O、二酸化炭素 CO₂、硫化鉄 FeS)

分解と化合の関係は次のように整理できます。

⇄ 分解と化合は逆の反応!
  • 分解: 1種類の物質 → 2種類以上の物質(例: 水 → 水素 + 酸素)
  • 化合: 2種類以上の物質 → 1種類の物質(例: 水素 + 酸素 → 水)
★ 覚えよう!
  • 化合 …… 2種類以上の物質が結びついて、別の1種類の物質ができる化学変化
  • 化合物 …… 化合によってできた、新しい物質
  • 分解と化合は 逆向き の関係

▲ 線香花火 ― 鉄粉が酸素と結びつく化合の例
📋 線香花火と化合

線香花火の火花は、火薬の中の 鉄粉 が空気中の 酸素 と結びつく化合の反応です。

鉄 + 酸素 → 酸化鉄

はげしく光や熱を出すので、これは「燃焼」(第4講で学習)でもありますが、もとはやはり化合の1つです。

✏️ 確認問題
問12種類以上の物質が結びついて、別の1種類の物質ができる化学変化を「化合」という。
○ 正解です。化合は 分解の逆 の反応です。できた物質を「化合物」といいます。
問2化合物は、もとの物質と同じ性質をもつ。
× 化合物は、もとの物質とは まったくちがう性質 をもつ別の物質です。たとえば、水素(燃える気体)+酸素(燃やす気体)→水(燃えない液体) のように、まったく別物になります。
問3水(H₂O)は、水素と酸素が化合してできた化合物である。
○ 正解です。水は2種類の元素(水素 H、酸素 O)からできているので化合物です。化学式 H₂O が「2種類の元素を含む」ことを表しています。
問4鉄(Fe)や酸素(O₂)は化合物のなかまである。
× 鉄も酸素も 1種類の元素 しかふくんでいないので、化合物ではなく 単体 です。化合物は「2種類以上」の元素からできている物質のこと。

    2鉄と硫黄の化合 ― 硫化鉄をつくる
    📖 説明
    0:00.0

    化合のうち、入試でいちばんよく出るのが 鉄と硫黄の反応 です。鉄粉と硫黄の粉末をよく混ぜて、試験管に入れて加熱します。鉄の元素記号は Fe、硫黄は S です(第2講で学習)。

    ▲ 試験管A(加熱前) → 試験管B(加熱後)
    📋 図のラベル
    A
    鉄と硫黄の混合物
    加熱前。鉄粉と硫黄の粉を混ぜただけ。
    B
    硫化鉄(FeS)
    加熱後。黒い1つの物質に変わった。

    鉄と硫黄を加熱すると、はげしく反応して光や熱を出し、黒い物質「硫化鉄」ができます。化学式で表すと FeS です。

    Fe + S → FeS(硫化鉄)

    大切なのは、反応の前と後で性質がまったく変わる ことです。下の表で比べてみます。

    調べ方 A 反応前(鉄と硫黄の混合物) B 反応後(硫化鉄)
    磁石を近づける くっつく(鉄が引きつけられる) つかない
    塩酸を加える 無臭 の 水素 が発生 刺激臭 の 硫化水素 が発生
    色 黒っぽい灰色 黒色のかたまり

    注) 硫化水素は 卵の腐ったようなにおい がします。温泉のにおいです。有毒なので、においをかぐときは手であおぐようにして少しずつ確かめます。

    このように、反応の前は鉄の性質(磁石につく、塩酸で無臭の気体が出る)が残っていますが、反応の後の硫化鉄は 磁石につかない、塩酸を加えると刺激臭のある気体(硫化水素)が出る、と性質がまるごと変わります。これは、もとの2つが結びついて、まったく別の1つの物質に変わったことを意味しています。

    ▲ 試験管の上部を加熱する
    📋 加熱のしかた

    鉄粉と硫黄の混合物は、試験管の 上部 に入れて加熱します。図のように、ガスバーナーの炎を試験管の上のほうに当てます。

    下のほうから加熱すると、はげしい反応の熱で試験管がわれてしまうことがあるからです(下の問1)。

    赤く反応が始まったら、ガスバーナーの火は消してかまいません。あとは反応自身が出す熱で次の部分が反応していきます(下の問2)。

    ⚠️ 実験上の重要ポイント

    問1) 鉄粉と硫黄の粉末の混合物を加熱する場合、上部を加熱するのはなぜですか。

    答) 激しく 反応するので、下部を加熱すると試験管がわれるからです。

    問2) どうしていったん赤くなると、火を消してもよいのですか。
    注) この反応は花火の導火線にも使われています。

    答) 発生した熱 が次の反応を起こすからです。

    ★ 覚えよう!
    • Fe + S → FeS(硫化鉄、黒色)
    • 反応前は 磁石につく、反応後は 磁石につかない
    • 反応前 + 塩酸 → 無臭 の 水素 が発生
    • 反応後 + 塩酸 → 刺激臭 の 硫化水素 が発生(卵の腐ったようなにおい・有毒)
    • 加熱は試験管の 上部 から(激しく反応するため)
    • 反応は 発生した熱 で続くので、火を消してもよい
    📚 語群 ― タップして選び、説明文の空欄をタップ!
    ✏️ 確認問題
    問1鉄と硫黄の混合物を加熱するとできる黒い物質は「硫化鉄(FeS)」である。
    ○ 正解です。Fe + S → FeS。これは入試でとてもよく出る化学反応です。
    問2反応後の硫化鉄に磁石を近づけると、鉄と同じく引きつけられる。
    × 硫化鉄は 磁石につきません。化合によって性質が変わるからです。「鉄が中に入っているから磁石につくはず」と思いがちですが、別の物質になっているので、もとの鉄の性質はもう残っていません。
    問3鉄と硫黄の反応は、いったん始まるとガスバーナーの火を止めても進み続ける。
    ○ 正解です。化合のときに 熱が発生 し、その熱で次の部分の反応が始まるため、火を止めても反応がつづきます(発熱反応)。
    問4硫化鉄にうすい塩酸を加えると、においのない気体(水素)が発生する。
    × 硫化鉄に塩酸を加えると、卵が腐ったようなにおい のする気体(硫化水素)が発生します。においのない水素が発生するのは、反応 前 の鉄+硫黄の混合物に塩酸を加えたときです。

      3銅と硫黄の化合 ― 硫化銅をつくる
      📖 説明
      0:00.0

      銅と硫黄も、加熱すると結びついて新しい物質ができます。鉄+硫黄の反応とよく似ています。銅の元素記号は Cu、硫黄は S です。

      赤褐色をした銅線(電気コードの中身に使われる金属)を、加熱した硫黄の蒸気の中に入れると、はげしく反応して銅線が黒い物質に変わります。これが「硫化銅」、化学式で CuS です。

      ▲ 銅線を硫黄の蒸気の中に入れて加熱
      📋 図のラベル
      銅線
      Cu
      赤褐色の金属。電気コードに使う。
      硫黄
      S
      試験管の底で加熱して蒸気にする。
      硫黄の
      蒸気
      加熱されて気体になった硫黄。これに銅線をかざすと、銅線がはげしく反応して黒くなる。

      Cu + S → CuS(硫化銅)

      銅と硫化銅の性質を比べてみます。

      調べ方 反応前(銅 Cu) 反応後(硫化銅 CuS)
      色 赤褐色(光沢あり) 黒色
      曲げてみる しなやかに曲がる もろい・折れる
      電気を通すか 通す 通さない

      反応前の銅は 金属 としての性質(光沢、しなやか、電気を通す)を持っていましたが、反応後の硫化銅は色も性質もまったくちがう、別の物質に変わっています。これも、鉄+硫黄のときと同じく、典型的な「化合」の反応で、できた硫化銅は 化合物 です。

      このように、金属(鉄や銅) + 硫黄 → 硫化〜 という形が共通のパターンになっていることに気づきましょう。

      💡 共通パターン
      • 鉄(Fe) + 硫黄(S) → 硫化鉄(FeS)
      • 銅(Cu) + 硫黄(S) → 硫化銅(CuS)
      • どちらも金属の光沢が消え、もろくなり、電気を通さなくなる
      ★ 覚えよう!
      • Cu + S → CuS(硫化銅、黒色)
      • 銅は赤褐色の 金属(電気コードに使われる)
      • 反応後は もろくて折れる、電気を通さない
      • 金属 + 硫黄 → 硫化〜 のパターン
      📚 語群 ― タップして選び、説明文の空欄をタップ!
      ✏️ 確認問題
      問1銅と硫黄を加熱すると、黒色の硫化銅(CuS)ができる。
      ○ 正解です。Cu + S → CuS。鉄+硫黄の反応(Fe + S → FeS)と同じパターンです。
      問2反応後の硫化銅は、もとの銅と同じく電気を通す。
      × 硫化銅は 電気を通しません。化合によって、もとの金属の性質(光沢・しなやか・電気を通す)はすべて失われ、別の物質に変わるからです。
      問3鉄+硫黄も、銅+硫黄も、化合の反応である。
      ○ 正解です。どちらも「2種類の物質が結びついて、別の1種類の物質ができる」反応 = 化合 です。
      問4硫化銅(CuS)は単体である。
      × 硫化銅は、銅(Cu)と硫黄(S)の 2種類の元素 からできているので 化合物 です。1種類の元素だけからできているのが「単体」(例: 銅 Cu、硫黄 S)。
