単元末テスト:化学反応式 ★★★

入試レベルの問題に挑戦!各都道府県の高校入試で実際に出題された問題から構成しています。じっくり考えて解いてみましょう。

1大問1:酸化銅+炭素の還元 〈三重県〉

🏆 三重県 公立高校入試問題
酸化銅と炭素の粉末の混合物を試験管に入れガスバーナーで加熱すると、気体が発生し、石灰水が白くにごり、試験管内の物質が黒色から赤色に変化した。

✏️ 大問1(全3問)
問1この反応の化学反応式として正しいものはどれか。
正解はイ:2CuO + C → 2Cu + CO₂。
左辺:Cu 2個、O 2個、C 1個 / 右辺:Cu 2個、O 2個、C 1個 で完全に一致しています。
問2この反応で発生した気体の化学式として正しいものはどれか。
正解はイ:CO₂(二酸化炭素)。石灰水を白くにごらせる気体は二酸化炭素です。反応式の右辺にもCO₂と書かれています。
問3試験管 A の中で起きた化学変化について、正しい説明はどれか。
正解はウ。CuOから酸素がうばわれてCu(還元)、Cが酸素と結びついてCO₂(酸化)になります。還元と酸化は必ず同時に起こるのが鉄則です。
2大問2:酸化銀の加熱分解 〈愛媛県改〉

🏆 愛媛県 公立高校入試問題(改題)
黒色の酸化銀を試験管に入れて加熱すると、気体が発生し、試験管内には白色(磨くと銀色)の物質が残った。

✏️ 大問2(全3問)
問1酸化銀(Ag₂O)の加熱分解(酸化銀→銀+酸素)を表す化学反応式として正しいものはどれか。

正解:ウ:2Ag₂O → 4Ag + O₂

酸素は気体として分子(O₂)になるので、左辺はAg₂O 2個必要です。

  • 左辺:Ag 4個、O 2個
  • 右辺:Ag 4個、O 2個 → 一致!
問2発生した気体が酸素であることを確かめる方法として正しいものはどれか。
正解はア。酸素にはものを燃やすはたらき(助燃性)があります。イは二酸化炭素、ウは水素、エは水の性質です。
問3酸化銀の加熱分解は、1種類の物質が2種類以上の物質に分かれる反応なので「分解」である。
○ 正解です。Ag₂O(1種類)→ Ag + O₂(2種類)になるので分解です。加熱によって分解する反応は特に熱分解と呼びます。
3大問3:水の電気分解 〈標準入試問題〉

🏆 標準入試問題
水酸化ナトリウムをとかした水を電気分解装置に入れ、電流を流したところ、両極から気体が発生した。陽極では14cm³、陰極ではある体積の気体が集まった。

✏️ 大問3(全3問)
問1水の電気分解を表す化学反応式として正しいものはどれか。
正解はウ:2H₂O → 2H₂ + O₂。水素も酸素も気体は分子(H₂、O₂)で発生します。
左辺:H 4個、O 2個 / 右辺:H 4個、O 2個 で完全に一致。
問2陽極で14cm³の酸素が発生したとき、陰極で発生した水素の体積はいくらか。
(化学反応式 2H₂O → 2H₂ + O₂ から求める)

正解:エ:28 cm³

化学反応式 2H₂O → 2H₂ + O₂ より:

  • 水素(H₂):酸素(O₂) = 2:1 の体積比
  • 酸素 14cm³ なら、水素 = 14 × 2 = 28 cm³
問3水酸化ナトリウムを水にとかしたのはなぜか、正しいものはどれか。
正解はイ。純粋な水はほとんど電流を通しません。水酸化ナトリウムをとかすことで電流を流れやすくするのが目的です。
4大問4:鉄と硫黄の化合 〈標準入試問題〉

🏆 標準入試問題
鉄粉と硫黄の粉末をよく混ぜ合わせて試験管に入れ、上部を加熱した。色が赤くなりはじめたら加熱をやめたが、反応は試験管全体に広がっていき、黒色の物質ができた。

✏️ 大問4(全4問)
問1この反応を化学反応式で表すとどうなるか。
正解はア:Fe + S → FeS。鉄も硫黄も原子のまま反応するので、係数は1です。生成する硫化鉄(FeS)は黒色。
問2加熱をやめても反応が試験管全体に広がっていった理由として正しいものはどれか。
正解はウ。鉄と硫黄の化合は発熱反応。発生した熱が次の反応の熱源となるため、加熱を止めても反応が続きます。線香花火や花火の導火線にも利用されています。
問3反応後にできた硫化鉄(FeS)に塩酸を加えると、どのような気体が発生するか。
正解はエ:硫化水素(H₂S)。硫化鉄に塩酸を加えると、卵の腐ったようなにおい(腐卵臭)の硫化水素が発生します。温泉のにおいとしても知られています(有毒)。
※反応前の鉄+硫黄の混合物に塩酸を加えると、無臭の水素が発生します。
問4鉄と硫黄の化合では、酸素が必要なので「酸化」とも呼ばれる。
× 鉄と硫黄の化合では酸素は使わないので、酸化ではありません。化合の中で酸素と結びつく反応だけが「酸化」です。Fe + S → FeS は単なる化合(または硫化)です。
5大問5:化学反応式の総合問題

🏆 総合問題
第7講で学んだ重要な化学反応式9個のまとめテストです。

✏️ 大問5(全4問)
問1炭酸水素ナトリウムの加熱分解の化学反応式として正しいものはどれか。
正解はイ:2NaHCO₃ → Na₂CO₃ + CO₂ + H₂O。Naが2個必要なので左辺はNaHCO₃ 2個。
暗記法:「2人のナホコ(NaHCO₃)さんはナツコ(Na₂CO₃)さんにかわる」
問2次の化学反応式のうち、左辺と右辺の原子数が合っていないものはどれか。

正解:イ(これが誤り)

Mg + O₂ → MgO では:

  • 左辺:Mg 1個、O 2個
  • 右辺:Mg 1個、O 1個 → 酸素の数が合わない!

正しくは 2Mg + O₂ → 2MgO です。

問3「2CuO + C → 2Cu + CO₂」の反応で、左辺の原子の総数として正しいものはどれか。

正解:ウ:5個

  • 2CuO → Cu 2個 + O 2個 = 4個
  • C → 1個
  • 合計 = 4 + 1 = 5個

右辺(2Cu + CO₂)も Cu 2個 + C 1個 + O 2個 = 5個で一致しています。

問4化学反応式を作るときは、まず物質の化学式を並べてから、左辺と右辺の各原子の数が等しくなるように係数(化学式の前の数字)を調整する。
○ 正解です。化学反応式を作る基本ステップは:
① 物質の化学式を並べる → ② 左辺と右辺の各原子の数を一致させるように係数を調整 → ③ 完成。
このとき化学式の中の添字(小さい数字)は変えないのがポイント。係数だけで調整します。
