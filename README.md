# GAMBAシステム 中学理科 化学変化

## 📖 使い方

このフォルダに入っているファイルは、ブラウザ(Chrome、Edge、Safariなど)で動く学習Webアプリです。

### 起動方法
1. このフォルダ全体を、お使いのパソコン・タブレット・スマホに保存します
2. **`index.html`** をダブルクリックすると、目次ページが開きます
3. 目次から「第1講 酸化銀の熱分解」をクリックすると、学習を始められます

## 📁 ファイル構成

| ファイル名 | 内容 |
|---|---|
| `index.html` | 目次ページ |
| `ch01.html` | 第1講 酸化銀の熱分解(サンプル) |
| `app.css` | 全体のデザイン(スタイルシート) |
| `app.js` | 動作プログラム(JavaScript) |

## 🎮 機能

### ▼ チャレンジモード(穴埋め)
- 「▶ チャレンジ開始」を押すと、本文中の重要語句が画面下の語群に移動します
- 語群から語句をタップ → 本文の空欄をタップ で語句を入れていきます
- 「✓ 判定」ボタンで答え合わせ
- 全問正解で🎉花吹雪+ベストタイムをランキング保存
- 間違えた語句は語群に自動で戻ります

### ▼ 補助機能
- **タイマー**(0:00.0 形式で計測)
- **手本を見る**:途中でわからなくなったら正答を確認できます
- **リセット**:入力をすべてクリア
- **中断**:チャレンジをやめて元の画面に戻る
- **ランキング**:上位3位までベストタイムを記録(リセット可能)

### ▼ 練習問題(一問一答)
- テキスト入力で答えを書き、「答え合わせ」で正誤判定

### ▼ 確認問題・単元末テスト(○×形式)
- ボタンをタップして○か×かを選び、まとめて答え合わせ
- 単元末テストは高校入試の過去問から精選

### ▼ 音声解説
- 四国メタンによる音声解説テキストを掲載
- 「▶ 再生する」ボタン(現在は仮置き)— 本番では音声ファイルが流れる仕組み

## 🎨 デザイン

中学生向けに明るく親しみやすい色合いで作っています。
- メインカラー: オレンジ(明るく元気)
- アクセント: 青・緑・紫
- 重要語句は黄色マーカーで強調

## 📝 拡張について

- 新しい講(第2講・第3講…)を追加するには、`ch01.html` をコピーして内容を変更します
- `app.css` `app.js` は共通利用できます
- 音声ファイル(mp3など)は VOICEVOX 等で生成し、`<button class="voice-btn">` の動作部分を `audio` 要素に差し替えます
