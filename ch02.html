第2講 / 物質が結びつく変化

化合 ★★☆

2種類以上の物質が結びついて、別の1種類の新しい物質ができる「化合」を学びます。鉄と硫黄、銅と硫黄、鉄と酸素…身のまわりの「結びつく」反応を見ていきましょう!

1化合って何?分解と何が違う?
📖 説明
0:00.0

第1講では「分解」を学びました。1種類の物質が、2種類以上の別の物質に分かれる化学変化でしたね。たとえば、炭酸水素ナトリウムを加熱すると、炭酸ナトリウム+水+二酸化炭素 の3つに分かれました。

今回学ぶ「化合」は、分解とは反対の変化です。つまり、2種類以上の物質が結びついて、別の1種類の新しい物質ができる化学変化のことを「化合」といいます。

そして、化合によってできた新しい物質を「化合物」とよびます。

たとえば、鉄(Fe)と硫黄(S)を加熱すると、結びついて「硫化鉄(FeS)」という別の物質ができます。これが化合の代表例です。

分解と化合は、ちょうど反対の変化ですが、どちらも「化学変化」のひとつです。原子の組み合わせが変わって、もとの物質とはまったく性質の違う物質ができます。

  • 化合物 …… 化合によってできた新しい物質
  • 分解と化合は反対の変化(どちらも化学変化)
  • 例:鉄 + 硫黄 → 硫化鉄

問1化合とは、2種類以上の物質が結びついて別の1種類の物質ができる化学変化である。
○ 正解です。「2つ以上 → 1つ」になるのが化合のポイント。逆に「1つ → 2つ以上」になる分解とセットで覚えましょう。
問2化合では、もとの物質と同じ性質をもつ新しい物質ができる。
× 化合でできる物質(化合物)は、もとの物質とはまったく違う性質をもちます。たとえば、鉄+硫黄 → 硫化鉄では、磁石につかなくなり、塩酸を加えるとちがう気体が発生します。
問3次のうち、「化合」にあたる変化はどれか。
正解はイ。鉄と硫黄(2種類)が結びついて硫化鉄(1種類)ができる化合です。アとウは分解、エは状態変化(化学変化ではない)です。

🏆 ベストタイム

    2鉄と酸素の化合 ― スチールウールを燃やそう
    📖 説明
    0:00.0

    スチールウール(細い鉄のかたまり)を空気中でガスバーナーで加熱します。すると、金属光沢のあった灰色のスチールウールが、ぼろぼろの黒色のかたまりに変化します。

    これは、鉄と空気中の酸素が結びついて、別の物質である「酸化鉄」ができたためです。これも化合の例ですね。

    ▶ 反応前と反応後の比較

    反応前(スチールウール=鉄)と反応後(酸化鉄)では、性質が大きく違います。

    • 色:にぶい金属の灰色 → 黒色
    • 電流:通す → 通さない
    • 塩酸を加える:水素が発生する → 気体は発生しない
    • 手でもむ:弾力性がある → ぼろぼろにくずれる

    ▶ 質量はどう変わる?

    同じ質量のスチールウールを2つ用意し、片方だけを加熱して上皿てんびんで比べると、加熱した方が重くなります。

    これは、鉄に空気中の酸素が加わったためです。酸素の分だけ質量が増えています。

    ⚗️ 反応のしかた

    鉄 + 酸素 → 酸化鉄

    灰色 + (空気中の酸素) → 黒色のもろい固体

    ★ 覚えよう!
    • スチールウール(鉄)を加熱 → 黒色の酸化鉄ができる
    • 酸化鉄は電気を通さず、ぼろぼろにくずれる
    • 反応後は酸素の分だけ質量が増える
    • もとの鉄に塩酸を加えると水素が発生するが、酸化鉄では発生しない

    ✏️ ミニテスト(全3問)
    問1スチールウールを加熱したときにできる物質は何か。
    正解はイ:酸化鉄。鉄が空気中の酸素と結びついて、黒色の酸化鉄ができます。
    問2同じ質量のスチールウールを2つ用意し、片方だけを加熱すると、加熱した方の質量が大きくなる。
    ○ 正解です。鉄に空気中の酸素が加わるので、酸素の分だけ質量が増えます。
    問3反応前のスチールウールと、反応後の酸化鉄について、ちがいとして正しくないものはどれか。
    正解はエ(これが誤り)。逆です!反応前(鉄)に塩酸を加えると水素が発生します。反応後(酸化鉄)は、塩酸を加えても気体は発生しません。

      3鉄と硫黄の化合 ― 線香花火のしくみ
      📖 説明
      0:00.0

      鉄粉と硫黄の粉末をよく混ぜて、試験管の上部を加熱します。なぜ下部ではなく上部を加熱するのでしょうか?この反応はとても激しいので、下部から加熱すると試験管が割れる危険があるためです。

      加熱した部分が赤くなったら、すぐに加熱をやめます。すると、火を消しても反応で発生した熱によって反応がどんどん広がっていき、試験管全体が反応していきます。これは線香花火の導火線にも利用されているしくみです。

      反応が終わると、黒色の物質「硫化鉄(FeS)」ができています。

      ▶ 鉄と硫化鉄、どこが違う?

      反応前の鉄+硫黄の混合物と、反応後の硫化鉄を比べてみましょう。

      • 磁石を近づける:鉄+硫黄はくっつく / 硫化鉄はつかない
      • 塩酸を加える:鉄+硫黄は無臭の水素が発生 / 硫化鉄は卵が腐ったようなにおいの硫化水素が発生

      つまり、もとの鉄や硫黄の性質はすっかり消えて、まったく違う物質になっています。これが化合の特徴です。

      ⚗️ 化学反応式

      Fe + S → FeS

      鉄 + 硫黄 → 硫化鉄(黒色)

      ⚠️ 注意:硫化水素は有毒

      硫化水素は卵が腐ったようなにおいで、温泉のにおいでもあります。有毒なので、直接吸い込まないように、手であおぐようにしてかぎましょう。

      ★ 覚えよう!
      • 鉄 + 硫黄 → 硫化鉄(FeS)(黒色)
      • 反応は試験管の上部から加熱(下部だと割れる)
      • 磁石:鉄+硫黄はつく / 硫化鉄はつかない
      • 塩酸:鉄+硫黄は無臭の水素 / 硫化鉄は腐卵臭の硫化水素

      問1鉄粉と硫黄の粉末を加熱したときにできる物質は何か。
      正解はウ:硫化鉄(FeS)。鉄(Fe)と硫黄(S)が結びついた化合物で、黒色です。

        4銅と硫黄の化合 ― 銅線が黒くもろくなる
        📖 説明
        0:00.0

        家庭の電気コードに使われている赤褐色の銅線。電流をよく通し、しなやかに曲がりますね。これを硫黄の蒸気の中に入れて加熱すると、なんと銅線は黒色のもろい物質に変化します。

        これが「硫化銅(CuS)」です。鉄と硫黄の場合と同じように、銅と硫黄が結びついて新しい物質ができたのです。

        ▶ 銅と硫化銅、どこが違う?

        • 色:銅は赤かっ色 / 硫化銅は黒色
        • 電流:銅はよく通す / 硫化銅は通さない
        • 固さ:銅はしなやかに曲がる / 硫化銅はもろくて、曲げると折れる

        つまり、銅と硫化銅は、「色」「電流」「固さ」のすべてで性質が違います。これも化合の特徴です。

        ⚗️ 化学反応式

        Cu + S → CuS

        銅(赤褐色) + 硫黄 → 硫化銅(黒色・もろい)

        ▶ 化合のまとめ

        これまで見てきた化合の例をまとめてみましょう。

        • 鉄 + 酸素 → 酸化鉄(スチールウールの加熱)
        • 鉄 + 硫黄 → 硫化鉄
        • 銅 + 硫黄 → 硫化銅

        このうち、酸素と結びつく化合のことを、特に「酸化」とよびます。酸化は次の第3講で詳しく学びますよ!
